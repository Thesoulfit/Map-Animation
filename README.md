# Map-Animation
Map animation is a part of a map that tracks a particular number of buses on particular route in Boston. The buses are tracked through different coloured markers.

# How to run?
You have to download all 3 files present in the repository to run it. After the download just click the file index.html and it will open in the browser.

#Future improvements
In future i plan to track multiple buses on multiple routes making it a cluster.
